Plugin support for GLSL.


Features:

  • Identifiers References
  • Code completion
  • Refactoring
  • Formatter
  • Syntax highlighting
  • Go to / find usage
  • Documentation for GLSL builtin functions
  • Support inside HTML and JS strings
  • Macro support

Supported file extensions:

  • .glsl
  • .vert
  • .vsh
  • .tesc
  • .tese
  • .geom
  • .gsh
  • .frag
  • .fsh
  • .comp

Planned (but not yet supported):

  • Type checking
  • Extensions
