Plugin support for GLSL.


Features:

  • Identifiers References
  • Code completion
  • Refactoring
  • Formatter
  • Syntax highlighting
  • Go to / find usage
  • Documentation for GLSL builtin functions
  • Support inside HTML
  • Macro support

Supported file extensions:

  • .glsl
  • .vert
  • .vsh
  • .tesc
  • .tese
  • .geom
  • .gsh
  • .frag
  • .fsh
  • .comp

Planned (but not yet supported):

  • Type checking
  • Full directive support
  • Extensions

Known issues:

  • As it's pretty hard to predict all the possible valid variables that a shader can have, warnings/errors about undeclared variables are disabled in order to avoid incorrect warnings. For now, if the variable is not found, it will get no syntax coloring. So this is not to say that a white colored variable is not valid, only that it may not be found. In the future, when all possible variables will be collected, this (essential) feature will be activated.
  • It's also hard to collect all OpenGL extensions functions and behaviour. Therefore, their names cannot be resolved, which leads to some problems, and more specifically, the code analyser will falsely mark the extended name as not found, even though it was found. So when using extensions you should expect to see some errors.
  • In older IDE versions the plugin icon (next to files and inside popups) is not available
